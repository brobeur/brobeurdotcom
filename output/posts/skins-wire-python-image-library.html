Skins Wire: Python Image Library

Posted: 2013-12-10 12:22 [Source]

<html><body><p>Lately I have been having fun with the PIL (Python Image Library). I have been developing a script that opens a bunch of random images from a folder, then edits them (merging mostly), and saves them to a new folders. It is currently in two notebooks - the first (skins) opens up the files and saves them to an output folder. The second (wirePIL) opens up those images in the output folder, edits them again, and saves them.

I also have got text added to the images - I'm using 'A Film By William Mckee' for now, this could be changed to anything else (great for opening titles/credits).
